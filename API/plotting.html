act.plotting (act.plotting)¶

This module contains procedures for plotting ARM datasets.

common.parse_ax(ax) Parses the given matplotlib axis.
common.parse_ax_fig(ax, fig) Parses the given matplotlib axis and figure.
common.get_date_format(days[, …]) Returns the DateFormatter object to use for the given number of days.
TimeSeriesDisplay(arm_obj[, subplot_shape]) A class for handing the display of timeseries of ARM Datasets.
