act.corrections (act.corrections)¶

The procedures in this module contain corrections for various datasets.

ceil.correct_ceil(obj[, fill_value, var_name])

This procedure corrects ceilometer data by filling all zero and negative values of backscatter with fill_value and then converting the backscatter data into logarithmic space.

doppler_lidar.correct_dl(obj[, var_name, …])

This procedure corrects doppler lidar data by filling all zero and negative values of backscatter with fill_value and then converting the backscatter data into logarithmic space.

mpl.correct_mpl(obj[, co_pol_var_name, …])

This procedure corrects MPL data: 1.) Throw out data before laser firing (heights < 0).

raman_lidar.correct_rl(obj[, var_name, …])

This procedure corrects raman lidar data by filling all zero and negative values of backscatter with fill_value and then converting the backscatter data into logarithmic space.

ship.correct_wind(obj[, wspd_name, …])

This procedure corrects wind speed and direction for ship motion based on equations from NOAA tech.
