act.corrections (act.corrections)¶

The procedures in this module contain corrections for various ARM datasets.

ceil.correct_ceil(arm_obj[, fill_value])

This procedure corrects celiometer data by filling all zero and negative values of backscatter with fill_value and then converting the backscatter data into logarithmic space.
