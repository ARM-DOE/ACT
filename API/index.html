This shows the details of the API of the Atmospheric data Community Toolkit. Documentation of each procedure in each module is provided as a reference.

act.corrections

The procedures in this module contain corrections for various datasets.

act.discovery

This module contains procedures for exploring and downloading data on ARM Data Discovery.

act.io

This module contains procedures for reading and writing various ARM datasets.

act.plotting

This module contains classes for displaying data.

act.qc

This module contains procedures for working with QC information and for applying tests to data.

act.retrievals

This module contains various retrievals for datasets.

act.utils

This module contains the common procedures used by all modules of the ARM Community Toolkit.
