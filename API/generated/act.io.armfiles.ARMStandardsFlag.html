act.io.armfiles.ARMStandardsFlag¶

class act.io.armfiles.ARMStandardsFlag[source]¶

This class stores a flag that is returned by :ref:act.io.armfiles.check_arm_standards.

Examples

my_flag = act.io.armfiles.ARMStandardsFlag(
    act.io.armfiles.ARMStandardsFlag.OK)
assert my_flag.OK
Attributes

This flag is set if the dataset conforms to ARM standards.
