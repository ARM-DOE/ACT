act.utils.datetime_utils.dates_between¶

act.utils.datetime_utils.dates_between(sdate, edate)[source]¶

Ths procedure returns all of the dates between sdate and edate.

Parameters
sdate: str

The string containing the start date. The string is formatted YYYYMMDD.

edate: str

The string containing the end date. The string is formatted YYYYMMDD.

Returns
all_dates: array of datetimes

The array containing the dates between sdate and edate
