act.io.armfiles.check_arm_standards¶

act.io.armfiles.check_arm_standards(ds)[source]¶

Checks to see if an xarray dataset conforms to ARM standards.

Parameters:
ds: xarray dataset

The dataset to check.
Returns:
flag: ARMStandardsFlag

The flag corresponding to whether or not the file conforms to ARM standards.
