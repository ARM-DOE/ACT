act.plotting.HistogramDisplay.HistogramDisplay¶

class act.plotting.HistogramDisplay.HistogramDisplay(obj, subplot_shape=(1, ), ds_name=None, **kwargs)[source]¶

This class is used to make histogram plots. It is inherited from Display and therefore contains all of Display’s attributes and methods.

Examples

To create a TimeSeriesDisplay with 3 rows, simply do:

ds = act.read_netcdf(the_file)
disp = act.plotting.HistogramDisplay(
   ds, subplot_shape=(3,), figsize=(15,5))

The HistogramDisplay constructor takes in the same keyword arguments as plt.subplots. For more information on the plt.subplots keyword arguments, see the matplotlib documentation. If no subplot_shape is provided, then no figure or axis will be created until add_subplots or plots is called.
