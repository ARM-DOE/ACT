act.qc.arm.add_dqr_to_qc¶

act.qc.arm.add_dqr_to_qc(obj, variable=None, assessment='incorrect,suspect', exclude=None, include=None)[source]¶

Function to query the ARM DQR web service for reports and add as a qc test. See online documentation from ARM Data Quality Office on the use of the DQR web service

https://code.arm.gov/docs/dqrws-examples/wikis/home

Parameters

  • obj (xarray Dataset) – Data object

  • variable (string or list) – Variables to check DQR web service for

  • assessment (string) – assessment type to get DQRs for

  • exclude (list of strings) – DQRs to exclude from adding into QC

  • include (list of strings) – List of DQRs to use in flagging of data

Returns

obj – Data object

Return type

xarray Dataset
