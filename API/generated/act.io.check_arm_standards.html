act.io.check_arm_standards¶

act.io.check_arm_standards(ds)[source]¶

Checks to see if an xarray dataset conforms to ARM standards.

Parameters

ds (xarray dataset) – The dataset to check.

Returns

flag (int) – The flag corresponding to whether or not the file conforms to ARM standards. Bit packed, so 0 for no, 1 for yes
