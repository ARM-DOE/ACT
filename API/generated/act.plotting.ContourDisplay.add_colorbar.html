act.plotting.ContourDisplay.add_colorbar¶

ContourDisplay.add_colorbar(mappable, title=None, subplot_index=(0,))¶

Adds a colorbar to the plot.

Parameters

  • mappable (matplotlib mappable) – The mappable to base the colorbar on.

  • title (str) – The title of the colorbar. Set to None to have no title.

  • subplot_index (1 or 2D tuple, list, or array) – The index of the subplot to set the x range of.

Returns

cbar (matplotlib colorbar handle) – The handle to the matplotlib colorbar.
