act.qc.QCFilter.unset_test¶

QCFilter.unset_test(var_name=None, qc_var_name=None, index=None, test_number=None, flag_value=False, flag_values_reset_value=0)[source]¶

Method to unset a test/filter from a quality control variable.

Parameters

  • var_name (str or None) – Data variable name.

  • qc_var_name (str or None) – Quality control variable name. Ignored if var_name is set.

  • index (int or list or numpy array) – Index to unset test in quality control array. If want to unset all values will need to pass in index of all values.

  • test_number (int) – Test number to remove.

  • flag_value (boolean) – Switch to use flag_values integer quality control.

  • flag_values_reset_value (int) – Value to use when resetting a flag_values value to not be set.

Examples

ds_object.qcfilter.unset_test(var_name, index=0, test_number=2)
