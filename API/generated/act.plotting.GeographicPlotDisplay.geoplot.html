Creates a latttude and longitude plot of a time series data set with data values indicated by color and described with a colorbar. Latitude values must be in degree north (-90 to 90) and longitude must be in degree east (-180 to 180).

  • lat_field (str) â Name of latitude field in object to use.

  • lon_field (str) â Name of longitude field in object to use.

  • dsname (str or None) â The name of the datastream to plot. Set to None to make ACT attempt to automatically determine this.

  • cbar_label (str) â Label to use with colorbar. If set to None will attempt to create label from long_name and units.

  • title (str) â Plot title.

  • projection (cartopy.crs object) â Project to use on plot. See https://scitools.org.uk/cartopy/docs/latest/crs/projections.html

  • plot_buffer (float) â Buffer to add around data on plot in lat and lon dimension.

  • stamen (str) â Dataset to use for background image. Set to None to not use background image.

  • tile (int) â Tile zoom to use with background image. Higer number indicates more resolution. A value of 8 is typical for a normal sonde plot.

  • cmap (str) â Color map to use for colorbar.

  • text (dictionary) â Dictionary of {text:[lon,lat]} to add to plot. Can have more than one set of text to add.

  • gridlines (boolean) â Use latitude and longitude gridlines.

  • **kwargs (keyword arguments) â Any other keyword arguments that will be passed into matplotlib.pyplot.scatter() when the figure is made. See the matplotlib documentation for further details on what keyword arguments are available.
