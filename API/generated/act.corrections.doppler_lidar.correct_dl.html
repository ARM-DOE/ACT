act.corrections.doppler_lidar.correct_dl¶

act.corrections.doppler_lidar.correct_dl(obj, var_name='attenuated_backscatter', range_normalize=True, fill_value=1e-07)[source]¶

This procedure corrects doppler lidar data by filling all zero and negative values of backscatter with fill_value and then converting the backscatter data into logarithmic space. Will also range normalize the values by multipling by the range^2 with default units.

Parameters

  • obj (Dataset object) – The doppler lidar dataset to correct. The backscatter data should be in linear space.

  • var_name (str) – The variable name of data in the object.

  • range_normalize (bool) – Option to range normalize the data.

  • fill_value (float) – The fill_value to use. The fill_value is entered in linear space.

Returns

obj – The doppler lidar dataset containing the corrected values.

Return type

Dataset object
