act.qc.qcfilter.parse_bit¶

act.qc.qcfilter.parse_bit(qc_bit)[source]¶

Given a single integer value, return bit positions.

Parameters

qc_bit (int or numpy int) – Bit packed integer number to be parsed.

Returns

bit_number – Array containing all bit numbers of the bit packed number. If no bits set returns empty array.

Return type

numpy.int32 array

Examples

> parse_bit(7) array([1, 2, 3])
