act.utils.ChangeUnits¶

class act.utils.ChangeUnits(xarray_obj)[source]¶

Class for updating units in the object. Data values and units attribute are updated in place. Coordinate variables can not be updated in place. Must use new returned dataset when updating coordinage varibles.

change_units([variables, desired_unit, ...])

Parameters

  • variables (None, str or list of str) -- Variable names to attempt to change units.
