act.qc.qcfilter.QCFilter.remove_test¶

QCFilter.remove_test(var_name, test_number=None, flag_value=False, flag_values_reset_value=0)[source]¶

Method to remove a test/filter from a quality control variable.

Parameters

  • var_name (str) – Data variable name.

  • test_number (int) – Test number to remove.

  • flag_value (boolean) – Switch to use flag_values integer quality control.

  • flag_values_reset_value (int) – Value to use when resetting a flag_values value to not be set.

Examples

> ds_object.qcfilter.remove_test(

var_name, test_number=3)
